# 特性

1. 下载口新增MOVES查询/MOVES=n写入切换次数n指令(0-2^32)
2. 删除04 05读写补偿指令
3. 增加写功能时长度匹配和范围匹配（根据表中变量长度及范围），写速度范围限制在20-200
4. 支持写03 38400波特率，修复写序列号5、切换次数4时参数长度不报错问题
5. 优化协议栈，定义异常码，使用宏优化语句
6. 支持0地址，支持AA广播地址查询阀组地址
7. 修复默认参数乱码：减速比4、速度20、电流0（2.6A）、波特率2（19200）、地址1、IO 0、原点补偿5、方向补偿0、通道数10、老化时间间隔5s
8. 异常报错代码：01功能码错、02操作码错、03写入参数范围超限/长度超限。04 05 06 0E 又存在，但理论上不出现在回复中
9. 修复09读写速度错误，速度修改为1个字节范围20-200
10. 序列号、切换次数必须手动清空，无法使用IIC清空
11. 最大通道数限制为16
12. 下载口参数增加范围限制 ADDR(0-63,64老化地址),CNT(3-16),POS(1,2),BAUD(1-3),SPD(20-200),INT(0-255),ISET(0-4),RDCR(1,4,10,16),HALF(0,1)
13. 64老化地址限制06写支持03读
14. 修复开启IO在B位置无法就近复位的情况
15. 新增调试口切换时间输出,添加调试信息输出开关
16. 调试口支持RST指令复位
17. LED闪烁:正常150ms,重试1100ms,异常500ms
18. 使用宏开关定义不同版本
19. 新增20减速比支持,20减速比速度限制在1到50,正常速度限制1到200
20. 下载口设置地址,通道数,波特率,速度,减速比,半通道增加默认值写入
21. 默认波特率改为9600
22. 下载口新增INSP点检模式，查询所有参数
23. 下载口新增REPLY设置回复方式，AGS新增0B回复方式设置，移动指令可以屏蔽回复
24. 中文显示所有指令和点检指令
25. C版本新增灯状态输出
26. 串口支援0D功能码读写半通道
27. 串口读指令新增长度限制
28. 支援modbus

# 版本号说明

## 命名方式
    vX.Y.Z-rN
  - X.Y.Z 为软件版本
  - N 为版次，需要根据此数字区别不同的修改
    eg. v1.3.1A-r1 版本 1.3.1A 版次 r1  类型A,IO
    v1.3.1-r11 版本 1.3.1 版次 r11     仅232/485
## 软件描述
  下载口会打印:
    程序版本  修改日期
    本软件描述 (控制方式)
    电路板版本  电路板描述 (最大电流)
```
 Version:v1.3.1A-r11(131A0011)  Time: Sep  4 2025 14:21:38             
 Description:Switch Valve (IO 232/485)            
 PCB:A12-909  Vertical Version (2.2A Max) 
```
