# 切换阀注射泵 TecanPump

- v1.3.1A-r3
- 2025.07.31


# 支持范围

- 232/485+IO
    - 901
    - 906
    - 909
- IO
    - 901
    - 906


# 特性

1. 下载口新增MOVES查询/MOVES=n写入切换次数n指令(0-2^32)
2. 删除04 05读写补偿指令
3. 增加写功能时长度匹配和范围匹配（根据表中变量长度及范围），写速度范围限制在20-200
4. 支持写03 38400波特率，修复写序列号5、切换次数4时参数长度不报错问题
5. 优化协议栈，定义异常码，使用宏优化语句
6. 支持0地址，支持AA广播地址查询阀组地址
7. 修复默认参数乱码：减速比4、速度20、电流0（2.6A）、波特率2（19200）、地址1、IO 0、原点补偿5、方向补偿0、通道数10、老化时间间隔5s
8. 异常报错代码：01功能码错、02操作码错、03写入参数范围超限/长度超限。04 05 06 0E 又存在，但理论上不出现在回复中
9. 修复09读写速度错误，速度修改为1个字节范围20-200
10. 序列号、切换次数必须手动清空，无法使用IIC清空
11. 最大通道数限制为16
12. 对地址(0-63)、波特率(1-3)、速度(20-200)、通道数(3-16)进行限制

# 版本号说明

## 命名方式
    vX.Y.Z-rN
  - X.Y.Z 为软件版本
  - N 为版次，需要根据此数字区别不同的修改
    eg. v1.3.1A-r1 版本 1.3.1A 版次 r1
    v1.3.1B-r2 版本 1.3.1B 版次 r2
## 软件描述
  下载口会打印: 
    程序版本 修改日期
    本软件描述 (协议)
    电路板版本 电路板描述 (最大电流)

```
 Version:V1.3.1A-r1(1311)  Time: Jul 21 2025 17:29:37
 Description:Switch Valve (232/485+IO)
 PCB:A12-906  Horizontal Version (2.5A Max)
```
